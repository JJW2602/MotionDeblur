## 3. Training Pipeline 구축

### 3.1 Dataset 준비

#### Train / Test Split
- GoPro blur-sharp 쌍 총 1029장
- **Train 900장 / Test 129장** (seed=42 shuffle)
- 저장: `dataset/gopro/split.json`

#### `GoProPairDataset` 클래스
- blur / sharp 이미지를 쌍으로 로드
- resolution 기반 리사이즈 (default 512, 64배수 정렬)
- canny edge map 동시 생성 (ControlNet 학습용)
- 반환: `{"blur", "sharp", "canny", "filename"}`

### 3.2 Training Architecture 선택 — `tuning_architecture`

config에서 `train.tuning_architecture`로 학습 대상을 선택:

| 값 | LoRA 대상 | 입력 조건 | 설명 |
|---|---|---|---|
| `SD3` | Transformer | sharp latent + noise | SD3 backbone을 deblur 태스크에 적응 |
| `controlnetSD3` | ControlNet | canny(blur) → transformer | ControlNet을 blur→sharp 매핑에 적응 |

### 3.3 학습 흐름 (공통)

```
1. Pipeline 로드 (SD3 or ControlNet+SD3)
2. 전체 freeze (VAE, Transformer, Text Encoders, ControlNet)
3. 선택된 모듈에 LoRA adapter 삽입
4. Prompt embedding 캐싱 (text encoder 학습 안 할 경우)
5. Dataset에서 blur/sharp 쌍 반복
   - sharp → VAE encode → target latent
   - noise 추가 → noisy latent 생성
   - [controlnetSD3] canny(blur) → ControlNet → block_hidden_states
   - Transformer forward → model_pred
   - loss = weighted MSE(model_pred, target)
   - backward → clip grad → optimizer step
6. LoRA weights를 safetensors로 저장
```

### 3.4 Config 구조 (`run.yaml`)

```yaml
train:
  tuning_architecture: SD3          # "SD3" | "controlnetSD3"
  output_dir: ./results/lora/sd3_img2img
  weight_name: sd3_lora.safetensors
  max_train_steps: 5000
  learning_rate: 1.0e-4
  rank: 8                           # LoRA rank
  alpha: 8                          # LoRA alpha
  gradient_checkpointing: true
  log_every: 50

  data:
    blur_dir: .../gopro/blur/images
    sharp_dir: .../gopro/sharp/images
    split_file: .../gopro/split.json
    resolution: 512
    num_workers: 4
```

### 3.5 실행 방법

```bash
# SD3 LoRA 학습
python run.py mode=train train.tuning_architecture=SD3

# ControlNet SD3 LoRA 학습
python run.py mode=train train.tuning_architecture=controlnetSD3
```
